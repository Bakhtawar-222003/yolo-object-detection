# YOLOv8 Object Detection

This project performs object detection using YOLOv8 and OpenCV.

## Technologies Used
- Python
- YOLOv8
- OpenCV
- Google Colab

## Project Files
- bakhtawar_yolo_project.py
- ssp0.1.jpg
- my_img_detector.jpg

## How It Works
1. Loads YOLOv8 model
2. Detects objects in image
3. Saves detected output image
4. Displays result using OpenCV

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python bakhtawar_yolo_project.py
```

## Output

Detected objects are saved in:
```text
my_img_detector.jpg
```
